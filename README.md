# CodeAlpha FAQ Bot

A simple **FAQ Chatbot** built with **Python, SpaCy, and Tkinter**.  
The bot uses **natural language processing (NLP)** and **cosine similarity (TF-IDF)** to match user questions with the most relevant FAQ answer.
